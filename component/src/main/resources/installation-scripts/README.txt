Instructions for usage

This guide contains instructions to install the FTP ballerina package into your ballerina installation. This is a one-time task.
Once done, you can then use this package across all your ballerina projects.

In this directory, you can find two scripts of interest for you. Please note that when running the following scripts,
you will need write permission to ballerina installation location.

1. install.{sh/bat} - Installs the FTP package.
2. uninstall.{sh/bat} - Uninstalls the FTP package if installed already.
